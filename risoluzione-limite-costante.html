Risoluzione: Limiti di una costante

Comprensione e dimostrazione del limite di una funzione costante

Esempio:

\[ \lim_{x \to 2} 3 \]

In questo esempio, calcoliamo il limite di una funzione costante \(f(x) = 3\) quando \(x\) tende a 2.

Risoluzione:

1
Definizione di limite di una costante

Per una funzione costante \(f(x) = c\), dove \(c\) è un numero reale, il limite di \(f(x)\) quando \(x\) tende a qualsiasi valore \(a\) è sempre uguale alla costante \(c\).

\[ \lim_{x \to a} c = c \]

Questo accade perché il valore della funzione \(f(x) = c\) rimane sempre costante, indipendentemente dal valore di \(x\).

2
Dimostrazione formale

Per dimostrare formalmente che \(\lim_{x \to a} c = c\), dobbiamo utilizzare la definizione ε-δ di limite:

Per ogni \(\varepsilon > 0\), dobbiamo trovare un \(\delta > 0\) tale che:

\[ 0 < |x - a| < \delta \Rightarrow |f(x) - c| < \varepsilon \]

Nel nostro caso, \(f(x) = c\), quindi:

\[ |f(x) - c| = |c - c| = 0 < \varepsilon \]

Questa condizione è sempre soddisfatta, indipendentemente dal valore di \(x\). Quindi, qualsiasi valore di \(\delta > 0\) funzionerà, poiché \(|f(x) - c| = 0\) è sempre minore di qualsiasi \(\varepsilon > 0\).

3
Applicazione all'esempio specifico

Nel nostro esempio, abbiamo \(f(x) = 3\) e vogliamo calcolare \(\lim_{x \to 2} 3\).

Applicando il principio del limite di una costante, abbiamo:

\[ \lim_{x \to 2} 3 = 3 \]

Questo significa che quando \(x\) si avvicina a 2, il valore della funzione \(f(x) = 3\) rimane costantemente uguale a 3.

Risultato finale:

\[ \lim_{x \to 2} 3 = 3 \]

Il limite di una funzione costante è sempre uguale al valore della costante stessa, indipendentemente dal punto in cui si calcola il limite.
