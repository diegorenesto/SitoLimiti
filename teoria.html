Il Concetto di Limite

Comprendi le basi e le applicazioni dei limiti matematici

Siano \(x_0\) ed \(l \in \mathbb{R}\).

Definizione di limite

Allora:

  • Un intorno di l \( \) è un intervallo aperto che contiene \(l\), cioè un intervallo del tipo \((l - \varepsilon, l + \varepsilon)\), dove \(\varepsilon > 0\). → In pratica, è "tutto ciò che sta vicino" a \(l\), distante meno di \(\varepsilon\).
  • Allo stesso modo, un intorno di x₀ \( \) è un intervallo del tipo \((x_0 - \delta, x_0 + \delta)\), dove \(\delta > 0\). → Quindi è "tutto ciò che sta vicino" a \(x_0\), distante meno di \(\delta\).

Questa idea degli intorni serve per scrivere in modo rigoroso la definizione di limite.

In parole semplici: La funzione \(f(x)\) ha limite \(l\) per \(x \to x_0\) se, facendo \(x\) sufficientemente vicino a \(x_0\) (cioè dentro un intorno di \(x_0\), tranne eventualmente il punto \(x_0\) stesso), il valore di \(f(x)\) rimane vicino a \(l\) (cioè dentro un intorno di \(l\)).

Definizione formale

\[\forall \varepsilon > 0 \; \exists \delta > 0 \; \text{tale che} \; x \in (x_0 - \delta, x_0 + \delta) \setminus \{x_0\} \Rightarrow f(x) \in (l - \varepsilon, l + \varepsilon)\]

Per ogni numero positivo \(\varepsilon\) (anche piccolissimo), cioè per ogni "tolleranza" che vogliamo avere rispetto al valore limite \(l\),

esiste un numero positivo \(\delta\) (anche molto piccolo), che rappresenta "quanto vicino" dobbiamo prendere i valori di \(x\) intorno a \(x_0\), tale che: ogni volta che prendiamo un valore di \(x\) dentro l'intervallo \((x_0 - \delta, x_0 + \delta)\) ma diverso da \(x_0\) (quindi ci avviciniamo ma non lo tocchiamo), allora il valore di \(f(x)\) starà sicuramente dentro l'intervallo \((l - \varepsilon, l + \varepsilon)\), cioè sarà "vicino" a \(l\).

In pratica: Quando diciamo che il limite di \(f(x)\) per \(x \to x_0\) è \(l\), stiamo dicendo che:

Possiamo rendere f(x) quanto vogliamo vicino a l, semplicemente scegliendo x abbastanza vicino a x₀ (ma senza che sia proprio uguale a x₀).

La regola è:

Più piccolo scegliamo \(\varepsilon\) → più piccolo dovremo scegliere \(\delta\), ma la cosa importante è che questo \(\delta\) esiste sempre.

Esempi di limiti fondamentali

Limite di una costante

\[ \lim_{x \to a} c = c \]

Dimostrazione:

Per ogni ε > 0, dobbiamo trovare δ > 0 tale che:

0 < |x - a| < δ ⇒ |c - c| < ε

Ma: |c - c| = 0 < ε (sempre vero, qualunque sia ε > 0)

Conclusione: Qualsiasi δ > 0 va bene.

Limite della funzione identità

\[ \lim_{x \to a} x = a \]

Dimostrazione:

Per ogni ε > 0, dobbiamo trovare δ > 0 tale che:

0 < |x - a| < δ ⇒ |x - a| < ε

Questa è direttamente verificata se scegliamo δ = ε.

Limite di un polinomio (caso: x²)

\[ \lim_{x \to a} x^2 = a^2 \]

Dimostrazione:

Per ogni ε > 0, dobbiamo trovare δ > 0 tale che:

0 < |x - a| < δ ⇒ |x² - a²| < ε

Si ha: |x² - a²| = |x - a| ⋅ |x + a|

Poniamo |x - a| < 1; allora: |x + a| ≤ |x - a| + 2|a| < 1 + 2|a|

Vogliamo: |x - a| ⋅ |x + a| < ε

Quindi, scegliamo δ = min {1, ε / (1 + 2|a|)}.

Limite della radice quadrata

\[ \lim_{x \to a} \sqrt{x} = \sqrt{a} \quad (a > 0) \]

Dimostrazione:

Per ogni ε > 0, dobbiamo trovare δ > 0 tale che:

0 < |x - a| < δ ⇒ |\sqrt{x} - \sqrt{a}| < ε

Usando l'identità: |\sqrt{x} - \sqrt{a}| = \frac{|x - a|}{|\sqrt{x} + \sqrt{a}|}

Per x vicino a a > 0, \sqrt{x} + \sqrt{a} ≥ 2\sqrt{a}

Quindi: |\sqrt{x} - \sqrt{a}| ≤ \frac{|x - a|}{2\sqrt{a}}

Scegliendo δ = 2ε\sqrt{a}, otteniamo il risultato desiderato.

Vai all'esercizio

Limite notevole del logaritmo

\[ \lim_{x \to 0} \frac{\ln(1 + x)}{x} = 1 \]

Dimostrazione:

Poniamo f(x) = \frac{\ln(1 + x)}{x} per x ≠ 0 e f(0) = 1

Per x ≠ 0, derivando ln(1+x), otteniamo \frac{1}{1+x}

Quando x tende a 0, \frac{1}{1+x} tende a 1

Quindi il limite è uguale a 1
